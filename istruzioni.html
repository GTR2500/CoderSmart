Generatore Istruzioni Dati

Come si usa

  1. Apri Dati e compila/aggiorna i 4 cataloghi: Famiglie, Materiali, Finiture, Tipi.
    • Salva memorizza in locale (browser).
    • Reset default ricarica i file in assets/data/.
    • Esporta/Importa JSON per backup o condivisione.
    • Salva su GitHub (con PAT) aggiorna i file nel repo.
  2. Vai su Generatore, scegli i valori:
    • Le caselle D/L/H/Sp/Mat/Fin/Tok sono abilitate in base alla Famiglia.
    • Solo i campi spuntati entrano nello ShortCode (gli altri diventano 000).
  3. Copia gli output:
    • ShortCode (auto) – solo codice.
    • Descrizione (auto) – formato compatto come richiesto.
    • Nome cartella (auto) – ShortCode + descrizione.

Struttura codice

FAM-TIPO-D3-L4-H4-S3-MAT-FIN-TOK — i campi non inclusi diventano 000.

  • FAM: codice famiglia (es. ARG)
  • TIPO: codice tipo (es. TUB)
  • D3/L4/H4/S3: misure con padding (es. D050, L0050)
  • MAT/FIN: codici (es. S235, V)
  • TOK: fino a 5 caratteri alfanumerici

Consigli

  • Usa codici brevi e univoci.
  • Valida prima di salvare su GitHub.
  • Fai un export JSON periodico per backup.
